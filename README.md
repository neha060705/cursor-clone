# Cursor Landing Page Clone
This is basic UI interface created using HTML, CSS



# Sections Implemented

1. Navbar
   -main logo
   -Logo, navigation links, and action buttons (Sign in / Download) 
   

3. Hero Section
   - Main headline introducing Cursor
   - Primary download button

4. Preview Section
   - Large product preview image
   - Trust message for developers

5. Company Logo Grid
   - Grid layout showcasing companies using Cursor

6. Feature Sections
   - Three feature blocks explaining:
     - AI Agent coding
     - Smart autocomplete
     - Cursor ecosystem
   - Alternating text and image layout

7. Testimonials Section
   - User reviews and quotes
   - Card-based grid layout with author details

8. Feature Cards
   - Three informational cards
   - Image + text + links

9. Changelog Section
   - Version updates with dates
   - Grid-based release notes


10. Recent Highlights
    - Blog, research, and customer updates

12. Download Call-to-Action
    - Large text and download button

13. Footer
    - Product, Resources, Company, Legal, and Social links



# Fonts Used
"CursorGothic", "CursorGothic Fallback", system-ui, Helvetica Neue, Helvetica, Arial, sans-serif;

# Colors Used

- Background
  - Black (`#000000`)
  - Dark brown/black (`rgb(27, 25, 19)`)
  - Dark gray (`#151515`)

- Text
  - White (`#ffffff`)
  - Light gray (`rgb(232, 228, 228)`)
  - Muted gray (`rgb(158, 153, 153)`)

- Accent Color
  - **OrangeRed** (used for links, highlights, and CTAs)




#Purpose

- Practice modern UI layout
- Improve CSS Grid & Flexbox skills
- Clone a real-world product landing page design
